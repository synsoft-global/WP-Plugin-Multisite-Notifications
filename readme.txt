== Description ==

Add a status message to your dashboard.


== Frequently Asked Questions ==

= Why should I choose WP admin status message plugin? =

WP admin status message adds a cool status to your dashboard. See your status or favorite quote always while managing your WordPress site or sites.


== Installation ==

1. Upload the `wp-admin-status-message` plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure and manage using the `Status Message Options` submenu item under Tools in the WordPress dashboard.
4. In the multisite configuration overrite all sites status message from `Status Message Options` submenu item under Settings in the WordPress Network dashboard.
5. Done!
